Expand description

Combat state change.

Variants

None

Not used, different kind of event.

EnterCombat

Source entity entered combat.

Destination contains the subgroup.

ExitCombat

Source entity left combat.

ChangeUp

Source entity is now alive.

ChangeDead

Source entity is now dead.

ChangeDown

Source entity is now downed.

Spawn

Source entity is now in game tracking range.

Not used in realtime API.

Despawn

Source entity is no longer being tracked or out of game tracking range.

Not used in realtime API.

HealthUpdate

Source entity health change.

Destination contains percentage as percent * 10000. For example 99.5% will be 9950.

Not used in realtime API.

LogStart

Logging has started.

Value contains the server Unix timestamp as u32. Buff damage contains the local Unix timestamp.

Source id is 0x637261 (ArcDPS id) if log EVTC and species id if realtime API.

LogEnd

Logging has ended.

Value contains the server Unix timestamp as u32. Buff damage contains the local Unix timestamp.

Source id is 0x637261 (ArcDPS id) if log EVTC and species id if realtime API.

WeaponSwap

Source entity swapped weapon set.

Destination contains the current set id. 0/1for underwater weapons and4/5` for land weapons.

MaxHealthUpdate

Source entity maximum health change.

Destination contains the new maximum health.

Not used in realtime API.

PointOfView

Source entity is “recording” player.

Not used in realtime API.

Language

Source entity contains the game text language.

Not used in realtime API.

GWBuild

Source entity contains the game build.

Not used in realtime API.

ShardId

Source entity contains the sever shard id.

Not used in realtime API.

Reward

Source entity got a reward chest.

Source is always self. Destination contains the reward id. Value contains the reward type.

BuffInitial

Appears once per buff per entity on logging start.

(statechange == 18 and buff == 18, normal combat event otherwise)

Position

Source entity position change.

Destination contains x/y/z as array of 3 floats.

Not used in realtime API.

Velocity

Source entity velocity change.

Destination contains x/y/z as array of 3 floats.

Not used in realtime API.

Facing

Source entity facing change.

Destination contains x/y as array of 2 floats.

Not used in realtime API.

TeamChange

Source entity team change.

Destination contains the new team id.

AttackTarget

Source entity is now an attack target.

Destination is the parent entity (gadget type). Value contains the current targetable state.

Not used in realtime API.

Targetable

Source entity targetability change.

Destination contains the new targetable state. 0 for no, 1 for yes. Default is yes.

Not used in realtime API.

MapId

Source entity contains the map id.

Not used in realtime API.

ReplInfo

Used internally by ArcDPS. Should not appear anywhere.

StackActive

Source entity with active buff.

Destination contains the stack id marked active.

StackReset

Source entity with reset buff.

Value is the duration to reset to (also marks inactive). Padding 61 contains the stack id.

Guild

Source entity is in guild.

Destination until buff damage is 16 byte (u8) guid.

Given in client form, needs minor rearrange for API form.

BuffInfo

Buff information.

If is_flanking probably invulnerable. If is_shields probably invert.

Offcycle contains the category. Padding 61 contains the stacking type. Padding 62 contains the probably resistance. Source master instance id contains the max stacks. Overstack value contains the duration cap.

Not used in realtime API.

BuffFormula

Buff formula.

Time contains type, attr1, attr2, param1, param2, param3, trait_src and trait_self as array of 8 floats. Source instance id contains buff_src and buff_self as array of 2 floats.

If is_flanking not NPC. If is_shields not player. If is_offcycle break.

overstack is value of type determined by padding 61.

Once per formula.

Not used in realtime API.

SkillInfo

Skill information.

Time contains recharge, range0, range1 and tooltiptime as array of 4 floats.

Not used in realtime API.

SkillTiming

Skill action.

Source contains the action. Destination contains at which millisecond.

One per timing.

Not used in realtime API.

BreakbarState

Source entity breakbar state change.

Value is u16 game enum (active, recover, immune, none).

Not used in realtime API.

BreakbarPercent

Breakbar percentage.

Value contains percentage as float.

Not used in realtime API.

Error

Error.

Time contains the error message as an array of up to 32 characters.

Not used in realtime API.

Tag

Source entity has tag.

Value is the id of the tag. Tag id is volatile, depends on game build.

BarrierUpdate

Source entity barrier change.

Destination contains percentage as percent * 10000. For example 99.5% will be 9950.

Not used in realtime API.

StatReset

Arc UI stats reset.

Source entity contains the npc id of the active log.

Not used in log EVTC.

Extension

Combat event with state change byte set to this.

ApiDelayed

Combat event with state change byte set to this.

Unknown

Unknown or invalid.
